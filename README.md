# Manifold Learning

Ce dépôt contient l'implémentation et l'analyse de diverses techniques d'**Apprentissage par Variétés**, ou, **Manifold Learning**. L'apprentissage par variétés est une forme de réduction de dimension non linéaire, où l'objectif est de découvrir les structures à basse dimension cachées dans des données de haute dimension.

## Aperçu du repo

Ce repo vise à explorer plusieurs algorithmes d'apprentissage par variétés et à comparer leurs performances sur différents jeux de données. L'apprentissage par variétés est particulièrement utile pour la visualisation des données, l'extraction de caractéristiques, et l'amélioration des performances des modèles d'apprentissage automatique sur des jeux de données de haute dimension.

### Techniques implémentées

Les techniques suivantes seront ou sont implémentées dans ce dépôt :

- **Analyse en Composantes Principales (ACP)** : Une méthode linéaire de réduction de dimension qui projette les données sur ses composantes principales.
- **Isomap** : Une technique de réduction de dimension non linéaire basée sur les distances géodésiques entre des points sur une variété.
- **Encodage Linéaire Local (LLE)** : Une méthode non linéaire qui calcule des représentations de faible dimension en préservant les relations locales entre les points.
- **t-Distributed Stochastic Neighbor Embedding (t-SNE)** : Une méthode non linéaire permettant de visualiser des données en haute dimension.
